# Vue3 SPA Login/Register with Laravel and Sanctum.

## Specification

- Laravel 9
- PHP 8.1
- Vue 3.2.31

## Installation

-  `https://github.com/skcals/laravel-vue-auth.git`
- `cd laravel-vue-auth/`
- `composer install`
- `npm install`
- `cp .env.example .env`
-  Update '.env' and set your database credentials
- `php artisan key:generate`
- `php artisan migrate`
- `php artisan serve`
- `npm run dev`
